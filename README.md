# Wanderlust – Short-Term Stay Booking Platform

Wanderlust is a full-stack web application that allows users to browse, list, and book short-term stays such as apartments, homestays, and vacation rentals. It includes authentication, property management, and booking features.

---

# Features

- User registration and login (authentication)
- Add, edit, and delete property listings
- Browse and search available stays
- Booking functionality for properties
- User dashboard for managing bookings
- Responsive UI for all devices

---

# Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
